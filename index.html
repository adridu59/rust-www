Rust
— a safe, concurrent, practical language.

fn main() {
  let st = ["World", "Moon"];

  for p in st.iter() {
    println!("Hello {}", p);
  }
}

Rust is a multi-paradigm and low-level language with native concurrency, safe memory access & data sharing that adds-in functional programming aspects and type classes.
